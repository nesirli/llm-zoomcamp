# Final Project

**[MetaScholar](https://github.com/nesirli/metascholar)** — RAG-based question answering over the metagenomics & microbiome literature. Ask a question, get an answer grounded in PubMed abstracts, with citations.

Built as a separate repo rather than nested here, so it can stand on its own as a portfolio project. See its README for architecture, roadmap, and setup instructions.

**Status:** 🚧 in progress — corpus fetch from PubMed is done (M0–M1); retrieval, generation, evaluation, and monitoring are next.
